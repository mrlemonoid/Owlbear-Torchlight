Torchlight Plugin

A simple torchlight and light effect plugin for Owlbear Rodeo.

This plugin lets you add customizable light sources to your scenes, such as torches, lanterns, windows, cages, and other atmospheric lighting effects. It is designed to help create a more immersive tabletop experience by making darkness, exploration, and mood lighting easier to manage.

Features
Add torchlight effects to your scene
Customize light size and appearance
Create different types of light sources
Useful for dungeons, night scenes, buildings, caves, and atmospheric maps
Lightweight and easy to use
Purpose

The goal of this plugin is to make lighting feel more alive and flexible during tabletop sessions, without making the setup complicated.
